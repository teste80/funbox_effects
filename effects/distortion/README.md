# Distortion

First effect inside `funbox_effects`.

## Controls

| Control | Function |
| --- | --- |
| Knob 1 | Drive |
| Knob 2 | Tone |
| Knob 3 | Level |
| Knob 4 | Blend |
| Knob 5 | Tightness |
| Knob 6 | Bias |
| Switch 1 | Left = soft, center = hard, right = asym |
| Switch 2 | Left = full, center = tight, right = thick |
| Switch 3 | Left = mono, center = stereo spread, right = dual mono |
| Dip 1 | Stereo input enable |
| FS 1 | Bypass |

## Build

From the repo root:

```sh
make EFFECT=distortion
```

Or from this directory:

```sh
make
```
